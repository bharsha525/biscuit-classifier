# biscuit-classifier
Build a Convolutional Neural Network (CNN) capable of automatically classifying biscuit wrappers into their respective brands. Provide a tool for market researchers to analyze design trends and consumer preferences. Enable packaging designers to draw inspiration from diverse wrapper styles.
